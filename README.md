## Project Descriptions

### AJAX
A simple project demonstrating how to load and display data dynamically using AJAX.

### Bootstrap Movie Site
A responsive movie website layout created using Bootstrap and basic JavaScript.

### Calculator
A basic calculator that performs arithmetic operations such as addition, subtraction, multiplication, and division.

### jQuery
A project that introduces basic jQuery syntax and DOM manipulation.

### Pizza Menu
An interactive pizza ordering menu that calculates the total price based on user selections.

### Countdown Slideshow
A slideshow combined with a countdown timer using JavaScript time functions.

### Expressions Alert
A beginner project using JavaScript expressions and alert messages.

### Functions
A project focused on creating and using JavaScript functions.

### Math Operators
Demonstrates basic mathematical operations in JavaScript.

### Dictionaries
Uses JavaScript objects as dictionaries to store and access data.

### Comparisons & Type Coercion
Explains comparison operators and type coercion in JavaScript.

### Ternary Operators & Constructors
Uses ternary operators and constructor functions to practice decision-making and object creation.

### Scope & Time Functions
Explores variable scope and time-based JavaScript functions.

### String Methods
Demonstrates common string methods such as slice, length, and replace.

### Loops & Arrays
Uses loops and arrays to process and display data.

### Tic-Tac-Toe Game
A classic two-player Tic-Tac-Toe game built with JavaScript.
